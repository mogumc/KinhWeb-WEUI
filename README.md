# KinhWeb-WEUI
A Template For KinhWeb

## 安装使用
  1.将文件放置到templates目录下  
  2.在``config.yaml``文件下将主题修改到weui  
  3.重启KinhWeb  

## 使用须知
本主题依赖KinhWeb 1.0.0开发,请保证KinhWeb版本≥1.0.0

## 功能
### V1.0.1版本
    1. 支持在线预览  
    2. 采用WeUI设计
